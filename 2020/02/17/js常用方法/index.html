push pop shift unshift 的区别

push()向Array的末尾添加若干元素，
pop()则把Array的最后一个元素删除掉

unshift()方法往Array的头部添加若干元素，
shift()方法则把Array的第一个元素删掉


    var arr = [1,2,3];
    arr.push(1);
    arr  //    [1,2,3,1]    
    arr.pop() ;
    arr // [1,2,3]  
    arr.unshift(1);
    arr // [1,1,2,3]
    arr.shift();
    arr // [1,2,3]
